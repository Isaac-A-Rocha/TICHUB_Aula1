# E-commerce Vue.js

Uma aplicação moderna de e-commerce construída com Vue.js 3, TypeScript e Options API.

## ✨ Funcionalidades

- **Catálogo de Produtos**: Exibição responsiva de produtos em cards modernos
- **Carrinho de Compras**: Sistema completo com adição e remoção de itens
- **Design Moderno**: Interface com gradientes, sombras e animações suaves
- **Responsivo**: Layout adaptável para desktop e mobile
- **TypeScript**: Tipagem completa em todo o projeto

## 🎨 Design Moderno

### Características Visuais
- **Gradientes**: Fundo com gradiente sutil e botões com efeito de brilho
- **Sombras**: Sistema de sombras em camadas para profundidade
- **Animações**: Hover effects suaves e transições elegantes
- **Tipografia**: Fonte Inter para melhor legibilidade
- **Cores**: Paleta moderna com variáveis CSS

### Componentes
- **ProductCard**: Cards elevados com hover effects e gradiente superior
- **Cart Summary**: Seção destacada com ícones e layout organizado
- **Botões**: Design consistente com feedback visual

## 🚀 Como Executar

```bash
npm install
npm run dev
```

Acesse `http://localhost:5177/` no navegador.

## 📱 Responsividade

- **Desktop**: Grid de 3 colunas para produtos
- **Tablet**: Grid adaptável
- **Mobile**: Layout single column com botões empilhados

## 🛠️ Tecnologias

- **Vue.js 3** com Options API
- **TypeScript** para tipagem
- **Vite** para desenvolvimento
- **CSS Moderno** com variáveis e gradientes
